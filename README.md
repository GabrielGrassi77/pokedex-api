# 🧭 Pokédex Web App

Uma aplicação web de Pokédex que consome a [PokéAPI](https://pokeapi.co) para buscar e exibir informações de Pokémon por nome ou ID.

---

## 🚀 Demonstração

Digite o nome ou ID de um Pokémon e clique em **Procurar** para ver suas informações.

---

## 🧠 Funcionalidades

- 🔎 Busca de Pokémon por nome ou ID  
- 🖼️ Exibição da imagem do Pokémon  
- 📛 Nome e ID do Pokémon  
- ⚠️ Mensagem de erro caso o Pokémon não seja encontrado  
- 🌐 Consumo de API em tempo real  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (Fetch API)  
- PokéAPI
